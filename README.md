## 📋 About

This is the source code for my personal portfolio website. It introduces who I am, what I do, and how to get in touch. Built with pure HTML and CSS — no frameworks, no dependencies, just clean and fast.

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Animated hero section
- Skills & tech stack showcase
- Work experience timeline
- About section
- Contact links

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, animations, grid, flexbox) |
| Fonts | Google Fonts — Syne, DM Mono, DM Sans |
| Hosting | Vercel |

No JavaScript frameworks. No build tools. No dependencies.

## 🚀 Getting Started

Clone the repo and open the file directly in your browser:

```bash
git clone https://github.com/procodeninja33/portfolio.git
cd portfolio
open index.html
```

That's it — no `npm install`, no build step required.
